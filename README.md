# AlexNet + GoogleNet finetune

## Getting started
### Instalation
#### Install anaconda
#### Create new environment
```
   conda create --name keras_theano
```
#### Install Theano
```commandline
pip install --user git+https://github.com/Theano/Theano.git
```

#### Install Keras v1.2 (not work with v2)
```
pip install keras==1.2
```
